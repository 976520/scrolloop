# 소개

scrolloop은 현대적인 웹 애플리케이션을 위한 고성능 가상 스크롤(Virtual Scrolling) 라이브러리입니다.

## scrolloop이란 무엇인가요?

scrolloop은 성능 저하 없이 수만 개의 데이터를 렌더링할 수 있게 해주는 경량 라이브러리입니다. **윈도잉(Windowing)** 또는 가상 스크롤 기술을 사용하여 사용자에게 보이는 항목만 렌더링함으로써 브라우저의 부하를 획기적으로 줄여줍니다.

## 설치하기

프로젝트 환경에 맞는 패키지를 설치하세요.

```bash
# React 프로젝트인 경우
npm install @scrolloop/react

# React Native 프로젝트인 경우
npm install @scrolloop/react-native
```
